YBB My Account mu-plugin 上传包

【用法】
1. SiteGround File Manager → public_html/wp-content/mu-plugins/
2. 上传本目录内：
   - ybb-my-account.php
   - ybb-my-account/ 整个文件夹
3. 勿把整个 siteground-upload-my-account 文件夹拖进 mu-plugins
4. SiteGround → Speed → Caching → Purge All
5. 验收：
   - /my-account/ 背景与首页一致（钓场图 + 白色蒙层）
   - 顶部横向站点导航 + 账户子菜单横向 pill
   - 页面 HTML 含 class ybb-account-page
